# 📊 AI on Tech Industry: Comprehensive Power BI Analytics Dashboard

"AI on Tech Industry" is a data visualization and macro-trend analysis project built entirely in Power BI. By consolidating four distinct, clean industry datasets from Kaggle spanning up to 2026, this dashboard provides a holistic view of the artificial intelligence revolution: tracking enterprise adoption, startup funding flows, global tech labor shifts (layoffs), and the evolving AI/Data science job market.

## 📌 Project & Data Overview
Instead of looking at isolated metrics, this project establishes a centralized data ecosystem by utilizing four structural datasets:
1. **`ai_company_adoption.csv`**: Measures enterprise-level indicator variables including AI integration stages, investments, and internal workforce adjustments.
2. **`startup_funding_dataset.csv`**: Tracks financial deployment, funding rounds, and valuations across emerging AI/Tech startups.
3. **`tech_layoffs_til_2025.csv`**: Documents macroeconomic restructuring across tech firms, monitoring workforce impacts by industry and geography.
4. **`global_ai_jobs.csv`**: Provides micro-level compensation metrics (salaries in USD), required technical skill sets, and work setups (Remote vs. Onsite) for data professionals through 2026.

---

## 🚀 Key Macro Insights Explored
* **The Structural Trade-Off:** The data illustrates a distinct shift where heavy legacy tech layoffs often run parallel to highly accelerated capital investments in enterprise AI automation.
* **Capital Density:** Funding rounds demonstrate an intense concentration of global venture capital pouring into core generative AI and automated infrastructure startups.
* **The Talent Premium:** While traditional tech sectors experience volatility, the data reveals robust salary resilience, compounding bonus distributions, and aggressive hiring demand for specialized machine learning and data engineering skill sets globally.

## 👨‍💻 How to View and Interact Locally
Since this dashboard contains sensitive enterprise modeling logic, it is hosted directly via its desktop source file:
1. Clone this repository to your local machine.
2. Ensure you have **Power BI Desktop** (Windows) installed.
3. Open the `AI on Tech Industry.pbix` file.
4. Interact freely with the dynamic slicers, drill-down maps, and cross-highlighting visual matrices.
